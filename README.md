This program tracks a patient's blood pressure over time and plots it onto a chart.
Additionally, it calculates the patient's average blood pressure and pulse in a given time frame.
The chart is generated dynamically based on the time frame.

HOW TO RUN:

- Clone the repo and navigate to Proj12 folder
- In command line, enter the following:
    python Proj12.py "Data.csv, (number of rows to plot)"
    - EXAMPLE: python Proj12.py "Data.csv, 31"
- Example outputs provided for 31, 61, 91, and 111 rows

- An image of the plot will automatically save to RequiredOutputZZZ.png
